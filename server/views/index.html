Motivation

Personalized applications are a two-edged sword: They assist users to keep the focus on relevant topics but they are often black boxes. In particular for research, transparency and provenance are essential. Therefore, we developed ScholarLensViz, a framework that allows the visualization of semantic user profiles based on a scholar's publications.

Learn more

About

ScholarLensViz provides different visualizations for semantic user profiles. Explanations for all profile entries are provided based on techniques from Natural Language Processing and Semantic Web. ScholarLensViz is the second component developed on top of the formely provided ScholarLens text mining pipeline that generates semantic user profiles.
